Introdução a HTML e CSS

Nesta seção, apresentaremos os conceitos básicos de HTML e CSS e sua importância no desenvolvimento web. HTML (HyperText Markup Language): HTML é a linguagem de marcação padrão usada para criar páginas da web. Ela é composta por elementos HTML que são usados para estruturar e apresentar o conteúdo na web. Os elementos HTML são marcados com tags e geralmente vêm em pares de abertura e fechamento, como <p>para parágrafos e <a> para links. Exemplo de código HTML simples:

CSS (Cascading Style Sheets): CSS é uma linguagem usada para estilizar documentos HTML. Ele permite controlar a aparência e o layout dos elementos HTML em uma página da web. Com CSS, você pode definir cores, fontes, margens, tamanhos e muito mais. Exemplo de código CSS simples (normalmente incluído em um arquivo separado):

Tags HTML

Aqui, exploraremos algumas das tags HTML mais comuns. utiliza várias tags para estruturar e formatar o conteúdo de uma página web. Aqui estão algumas das tags HTML mais comuns e seu propósito:

<!DOCTYPE html> Define a versão do HTML que está sendo usada. Deve ser a primeira linha em um documento HTML.

<html>: Define o elemento raiz de um documento HTML.

<head> Contém informações sobre o documento, como o título da página,meta informações e links para folhas de estilo externas.

<title> Define o título da página que aparece na aba do navegador ou na guia.

<meta> Fornece informações adicionais sobre o documento, como conjunto de caracteres, descrição e palavras-chave (para mecanismos de busca).

<link> Usado para vincular folhas de estilo externas ao documento HTML.

<style>Permite a definição de estilos CSS diretamente no documento HTML.

<body>Contém o conteúdo visível da página,como texto,imagens e outros elementos.

<h1><h2><h3><h4><h5><h6>São cabeçalhos que variam em tamanho e importância,sendo <h1>o maior e mais importante,e <h6> o menor.

<p>Define um parágrafo de texto.

<a>Cria um link para outra página ou recurso da web.

<img>Insere uma imagem na página.

<ul>Cria uma lista não ordenada.

<ol>Cria uma lista ordenada.

<li> Define um item de lista dentro de ul ou ol.

<div>É um elemento de divisão que é frequentemente usado para agrupar e estilizar conteúdo.

<span> É usado para aplicar estilos a parte de um texto ou conteúdo sem alterar a estrutura.

<table> Cria uma tabela.

<tr> Define uma linha em uma tabela.

<td>Define uma célula em uma tabela.

<th> Define um cabeçalho de célula em uma tabela.

<form>Cria um formulário para coletar informações do usuário.

<input>Define um campo de entrada em um formulário.

<button>Cria um botão em um formulário.

<textarea>Cria uma área de texto multilinha em um formulário.

<label> Define uma etiqueta para um campo de entrada em um formulário.

<select> Cria uma lista suspensa em um formulário.

<option> Define uma opção dentro de uma lista suspensa.

<iframe>Permite incorporar outra página web dentro da página atual.

<audio>Insere áudio na página.

<video>Insere vídeo na página. Essas são apenas algumas das tags HTML mais comuns.

Existem muitas outras tags disponíveis para diversos fins. A escolha das tags a serem usadas depende da estrutura e do conteúdo específico de cada página web.

CSS

Esta seção abordará os princípios do CSS (Cascading Style Sheets) para estilizar páginas web e melhorar sua aparência.
CSS é uma linguagem de estilo utilizada para controlar a apresentação e a formatação de documentos HTML. O princípio fundamental do CSS é separar o conteúdo (HTML) da apresentação (estilos). Isso permite que você defina como elementos HTML devem ser exibidos visualmente sem alterar a estrutura do documento HTML em si. Aqui estão alguns dos princípios básicos do CSS:
Seletores: Os seletores são usados para escolher elementos HTML aos quais você deseja aplicar estilos. Por exemplo, você pode selecionar todos os elementos <p> ou apenas um elemento com um determinado id ou class.
Propriedades: As propriedades são atributos que você pode definir para alterar a aparência dos elementos selecionados. Por exemplo, você pode definir a cor do texto, o tamanho da fonte, o espaçamento entre linhas e muito mais.
Valores: As propriedades recebem valores que determinam como elas devem ser aplicadas. Por exemplo, você pode definir a cor do texto como "vermelho", o tamanho da fonte como "16px" e assim por diante.
Declarações CSS: Uma declaração CSS consiste em um seletor e um conjunto de propriedades e valores.
Por exemplo:

p {
color: red;
font-size: 16px;
}

Neste exemplo, todas as tags <p>terão texto vermelho e uma fonte de 16 pixels.
Cascata: O "C" em CSS significa "cascading", o que significa que várias regras CSS podem ser aplicadas ao mesmo elemento, e o navegador determinará qual regra tem prioridade com base em uma hierarquia de especificidade. Isso permite que você crie estilos flexíveis e granulares.
Herança: Algumas propriedades são herdadas dos elementos pai para os elementos filhos. Por exemplo, se você definir a cor do texto em um elemento pai, os elementos filhos herdarão essa cor, a menos que sejam especificamente sobrescritos.
Unidades de medida: Você pode usar diferentes unidades de medida, como pixels (px), porcentagens (%), ems (em), entre outras, para dimensionar elementos e definir tamanhos de fonte.
Box Model: O modelo de caixa é fundamental para entender como o espaço é calculado para um elemento, incluindo seu conteúdo, margens, bordas e preenchimento.
Media Queries: Você pode usar media queries para aplicar estilos com base nas características do dispositivo, como largura de tela, resolução, orientação e muito mais, tornando seu design responsivo.
Estilos Externos e Internos: Você pode incluir estilos CSS em um documento HTML de várias maneiras, incluindo estilos internos (dentro de uma tag <style>) e estilos externos (em um arquivo .css separado).
